# Longest Substring Without Repeating Characters

**Difficulty:** Medium

**Topics:** Hash Table, String, Sliding Window

**Bookmark:**

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        i, j = 0, 0
        ss = set()
        l = 0

        while j < len(s):
            if s[j] in ss:
                ss.remove(s[i])
                i += 1
                continue

            ss.add(s[j])
            j += 1
            l = max(l, len(ss))

        return l
```
